
# Cybersecurity Compliance Checklist (SOC 2 • NIST CSF • CIS v8)

A unified compliance checklist designed for SMBs that need enterprise-grade security without unnecessary complexity.

---

## 1. Access Control (AC)

| Requirement | Status | Evidence |
|-------------|--------|----------|
| MFA enforced org-wide | ☐ | Policy screenshot |
| Least-privilege access model | ☐ | IAM review |
| Automated offboarding ≤ 24 hours | ☐ | HR + IT logs |
| Anomaly detection enabled | ☐ | SIEM reports |

---

## 2. Asset Management (AM)

| Requirement | Status | Evidence |
|-------------|--------|----------|
| Updated asset inventory | ☐ | Asset list |
| Cloud resource inventory | ☐ | AWS/Azure export |
| Unauthorized devices blocked | ☐ | NAC logs |

---

## 3. Data Protection (DP)

| Requirement | Status | Evidence |
|-------------|--------|----------|
| Data classification policy | ☐ | Documentation |
| Encryption in transit & at rest | ☐ | Config screenshots |
| Backup policy defined | ☐ | Backup strategy |
| Backup validation monthly | ☐ | Verification logs |

---

## 4. Logging & Monitoring (LM)

| Requirement | Status | Evidence |
|-------------|--------|----------|
| Centralized logging | ☐ | SIEM dashboard |
| Log retention compliant | ☐ | Storage configs |
| Critical alerts configured | ☐ | Alert rules |
| Log reviews scheduled | ☐ | Review notes |

## 5. Vulnerability Management (VM)

| Requirement | Status | Evidence |
|-------------|--------|----------|
| Weekly vulnerability scans | ☐ | Scan report |
| Patch cycle ≤ 30 days | ☐ | Patch logs |
| Critical patches ≤ 7 days | ☐ | Ticketing evidence |
